ECUADOR INFORMADO - SITIO WEB

1. Abre el archivo index.html en tu navegador.
2. Para publicar gratis:
   - Opción simple: subir carpeta a Netlify Drop.
   - Opción académica: subir archivos a GitHub Pages.
3. Para agregar videos:
   - Crea una carpeta llamada videos.
   - Coloca tus archivos MP4.
   - Reemplaza las tarjetas de la sección Videoteca por etiquetas <video controls>.
